def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

for i in range(5):
    print(f"Factorial of {i} is {factorial(i)}")

def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

print(is_prime(17))  # True
print(is_prime(18))  # False

Data science is the process of extracting insights and knowledge from structured and unstructured data using statistical and computational methods.

Machine learning is a subset of artificial intelligence that enables systems to learn from data and improve from experience without being explicitly programmed.

A common task in data science is data cleaning, which involves detecting and correcting errors or inconsistencies in data to improve its quality.

Python libraries such as pandas and numpy are widely used for data manipulation and numerical computations.

Pandas provides data structures like DataFrame, which allows for efficient handling of tabular data.

NumPy offers powerful N-dimensional array objects and functions for performing mathematical operations.

Scikit-learn is a popular Python library for implementing machine learning algorithms like classification, regression, and clustering.

A decision tree is a flowchart-like structure used for classification and regression tasks.

Data visualization tools such as Matplotlib and Seaborn help represent data graphically to identify trends and patterns.

The confusion matrix is a performance measurement for classification problems, showing true positives, false positives, true negatives, and false negatives.

Feature engineering is the process of selecting, modifying, or creating features from raw data to improve the performance of machine learning models.

Overfitting occurs when a model learns the training data too well, including noise, leading to poor generalization on new data.

Cross-validation is a technique for assessing how well a model generalizes by dividing data into training and testing sets multiple times.

Natural language processing (NLP) enables machines to understand and interpret human language using computational techniques.

Big data refers to extremely large datasets that require specialized tools and techniques to process efficiently.

Cloud computing platforms such as AWS, Azure, and Google Cloud provide scalable resources for data storage and computation.

Databases store data in structured formats; SQL is a language used to query and manipulate relational databases.

def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

for i in range(5):
    print(f"Factorial of {i} is {factorial(i)}")

def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

print(is_prime(17))  # True
print(is_prime(18))  # False

Machine learning is a subset of artificial intelligence that enables systems to learn from data and improve from experience without being explicitly programmed.

NumPy offers powerful N-dimensional array objects and functions for performing mathematical operations.

A decision tree is a flowchart-like structure used for classification and regression tasks.

Data visualization tools such as Matplotlib and Seaborn help represent data graphically to identify trends and patterns.

The confusion matrix is a performance measurement for classification problems, showing true positives, false positives, true negatives, and false negatives.

Feature engineering is the process of selecting, modifying, or creating features from raw data to improve the performance of machine learning models.

Overfitting occurs when a model learns the training data too well, including noise, leading to poor generalization on new data.

Cross-validation is a technique for assessing how well a model generalizes by dividing data into training and testing sets multiple times.

Natural language processing (NLP) enables machines to understand and interpret human language using computational techniques.

Big data refers to extremely large datasets that require specialized tools and techniques to process efficiently.
